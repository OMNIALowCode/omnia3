List of features available on OMNIA Platform 3.0:

## Modeler

### Entities

The following types of entities can be modeled:

    - Agents
    - Resources
    - Generic Entities
    - Commitments
    - Events
    - Documents
    - Series
    - Data Sources
    
Within entities modeling, it is possible to make the following operations:

    - Manage Attributes
    - Manage Behaviours
    - Review User Interface
